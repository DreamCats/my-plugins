# 任务模板

## 单个任务模板

**每个任务必须包含**：

````markdown
### [任务编号] [任务名称]

**描述**：[一句话描述任务内容]

**文件**：[仓库相对路径（以 Git worktree 根目录为准，禁止绝对路径）]

**参考（可选）**：

- 设计文档：`design.md` - [引用章节]

**输入**：[任务输入/依赖]
**输出**：[任务产出]

**代码框架**：

```typescript
// [代码框架]
````

**验证**：

- [ ] [验证条件 1]
- [ ] [验证条件 2]

**预计时间**：2-5 分钟
````
