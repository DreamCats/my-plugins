# 编译验证

## 原则

**只编译改动的包，禁止全量编译。**

```bash
# ✅ 正确：只编译改动的包
go build ./path/to/changed/package/...

# ❌ 错误：全量编译
go build ./...
```

## 编译失败处理

如果编译失败：

1. **语法错误** → 修复后重新编译
2. **依赖错误** → 检查 import 路径
3. **类型错误** → 检查类型定义

修复后重新验证编译。

## 验证清单

- [ ] 改动已完成
- [ ] 编译通过
- [ ] 代码格式正确（`gofmt`/`gopls format`）
